## Датасет

-   Загальна кількість прикладів: **6497**
-   Кількість ознак: **11**
-   Завдання: **бінарна класифікація** (якісне / неякісне вино)

## Моделі

### Рівень 1 --- Проста модель

### **Логістична регресія**

-   Точність: **0.8215**
-   Переваги: проста, швидка, інтерпретована
-   Недоліки: слабко працює з нелінійними залежностями
-   Проблема: модель пропускає багато позитивних зразків (низький recall
    класу 1)

------------------------------------------------------------------------

### Рівень 2 --- Модель середньої складності

### **Random Forest**

-   Точність: **0.8815**
-   Переваги: найкращий результат, добре працює з нелінійними даними,
    стійка
-   Недоліки: менш інтерпретована
-   Дає найвищий F1-score для обох класів

------------------------------------------------------------------------

### Рівень 3 --- Складна модель

### **Багатошаровий перцептрон (MLP)**

-   Точність: **0.8562**
-   Переваги: здатний моделювати складні патерни
-   Недоліки: чутливий до масштабування та параметрів
-   Поступається Random Forest, але значно краще за логістичну регресію

## Висновок

**Random Forest** показав найкращий результат серед трьох моделей.
- Логістична регресія добре класифікує основний клас, але має низьку
якість для менш представленого класу.
- MLP працює стабільно, але не перевершив Random Forest.

Загалом **Random Forest є оптимальним вибором** для цього датасету
